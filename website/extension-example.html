Parvoviridae-GLUE can provide the basis for in depth studies of specific viruses. For example

On computers with GLUE installed, the Parvoviridae-GLUE project can be instantiated by navigating to the project folder, initiating GLUE, and issuing the following command in the GLUE shell: 

  Mode path: /
  GLUE> run file parvoviridaeProject.glue

2. Normalising the coordinate space

An issue with comparative analysis of CRESS virus genomes is that many are circular, and the first nucleotide positions can differ so that sequences end up ‘staggered’ in an alignment.

GLUE can be used to normalise all sequences in a project to a given start position. There are also tools for dealing with reverse complemented sequences - also an issue in CRESS due to the ambisense nature fo the genomes.

Again this would be very valuable for any work relating to circular genomes (I know an Anellovirus researcher who had major issues with Genbank due to this).
